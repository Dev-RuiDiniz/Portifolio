## Executar localmente

```bash
python -m http.server 8000
```

Abra `http://localhost:8000`.

## Publicar no GitHub Pages

1. Crie um repositório público, por exemplo `portfolio`.
2. Envie os arquivos para a branch `main`.
3. Acesse **Settings → Pages**.
4. Em **Source**, escolha **Deploy from a branch**.
5. Selecione `main` e `/ (root)`.

A URL será semelhante a:

```text
https://dev-ruidiniz.github.io/portfolio/
```

Para usar a URL principal `https://dev-ruidiniz.github.io/`, o repositório deve se chamar exatamente `Dev-RuiDiniz.github.io`.
